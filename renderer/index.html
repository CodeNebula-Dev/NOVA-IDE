    SuperNova AI & Ollama Integration Guide

    SuperNova v1 Prompt Commands

    SuperNova v1 features a multi-agent orchestrator tailored for both chatting and building code:

    /build <instruction>

    Runs the multi-agent builder pipeline. SuperNova will automatically plan, view, edit, and verify file changes in your workspace.

    /chat <question>

    Runs single-agent conversational mode. Perfect for explaining code, debugging, or asking general questions without modifying files.

    Local Ollama Integration

    Run your AI completely local and offline for maximum privacy and performance. The SuperNova pipeline will bridge local Ollama models to read your files and make precise edits.

    Quick Setup Checklist:

    1. Download and install Ollama from ollama.com.
    2. Make sure Ollama is running in the background (default port 11434).
    3. Open the Ollama Status Panel on the left side Activity Bar and click Verify.
    4. Pull a recommended model, e.g. qwen2.5-coder:7b or llama3.1:8b.
    5. Switch the Preferred LLM Provider to Ollama in your Settings pane.
